To earn points, simply navigate to `Project -> Subject -> Group -> Users`

The table displays users that made progress in **the skill group** \- anyone that successfully reported and earned points for at least 1 skill that falls under this skill group\. Users' progress for **the skill group** is shown here as well. The progress shows:

* **% Complete** \- user's skill group completion percentage
* **Current Points** \- current points earned toward this skill group \(sum for all the skills under the skill group\)
* **Total Points** \- total available points for the skill group
* **Points First Earned** \- date when the user first earned points in this group
* **Points Last Earned** \- date when the user most recently earned points in this group
